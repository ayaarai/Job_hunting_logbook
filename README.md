# 企業選考ログ

転職活動の企業選考を日記のように記録するシンプルなWebアプリです。

## スクリーンショット

| 一覧画面 | 新規追加 | 編集画面 |
|---------|---------|---------|
| 企業カード一覧・検索 | フォーム入力 | 内容編集・削除 |

## 機能

- 企業選考の記録を追加・編集・削除
- 選考フェーズをカラーバッジで視覚化
- 企業名・フェーズ・応募経路でキーワード検索
- 応募経路が「エージェント」の場合のみエージェント名欄を表示
- `localStorage` によるデータ永続化（ページ再読み込み後も保持）
- 初回起動時はモックデータ3件を自動表示

## 使い方

ビルド不要。`index.html` をブラウザで開くだけで動作します。

```bash
open index.html
```

またはVS Codeの Live Server 拡張などで開いてください。

## データ構造

```json
{
  "id": "1710000000000",
  "company": "株式会社〇〇",
  "route": "エージェント",
  "agent": "リクルートエージェント",
  "stage": "1次面接",
  "interviewDate": "2026-03-10",
  "interviewer": "田中部長",
  "memo": "雰囲気は良かった。技術スタックの話が盛り上がった。",
  "createdAt": "2026-03-01T10:00:00Z"
}
```

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `id` | string | 一意なID（`Date.now()` で自動生成） |
| `company` | string | 企業名（必須） |
| `route` | string | 応募経路 |
| `agent` | string | エージェント名（応募経路が「エージェント」の場合） |
| `stage` | string | 選考フェーズ |
| `interviewDate` | string | 面接日（YYYY-MM-DD） |
| `interviewer` | string | 面接官名 |
| `memo` | string | メモ（自由入力） |
| `createdAt` | string | 作成日時（ISO 8601） |

## API仕様（将来のバックエンド連携用）

現在は `localStorage` で動作しますが、以下のREST APIに対応する設計になっています。

| メソッド | エンドポイント | 説明 |
|--------|-------------|------|
| GET | `/logs` | 一覧取得 |
| POST | `/logs` | 新規作成 |
| PUT | `/logs/{id}` | 更新 |
| DELETE | `/logs/{id}` | 削除 |

バックエンドに切り替える場合は `index.html` 内の `useLogs()` フックの `localStorage` 操作をfetch呼び出しに置き換えてください。
